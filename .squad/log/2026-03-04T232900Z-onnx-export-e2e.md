## Key Learnings

1. **Integration test gaps:** Tests that mock external dependencies (like Web Workers) don't surface routing bugs. Actual HTTP request flow validation needed.
2. **Path resolution:** Backend `AppContext.BaseDirectory` is binary output dir, not project root. Model files need explicit copy-to-output or absolute path config.
3. **Opset compatibility:** PyTorch's new dynamo exporter targets opset 18+. onnxruntime shape inference bug on opset 18 blocks quantization; fp32 fallback mitigates.
