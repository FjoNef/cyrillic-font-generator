# Directive Capture (2026-02-25T112635)

## User Directives (via FjoNef)

Resolved 4 open questions from architecture kickoff:

### 1. Cyrillic Scope
**Decision:** Russian only (66 chars: А–Я + а–я with Ё/ё) for v1
- Extended Cyrillic (Serbian, Ukrainian, Bulgarian) deferred to future releases
- Rationale: Focused MVP scope, proven training data availability

### 2. Quality Bar
**Decision:** MVP pipeline first — get end-to-end working, iterate quality later
- Prioritize functional delivery over aesthetic perfection
- Establish training → inference → font generation pipeline
- Quality refinement in post-MVP iterations

### 3. Hosting
**Decision:** Self-hosted (not Azure-managed services)
- ASP.NET Core backend serves static model + SPA
- No cloud infrastructure lock-in
- Lightweight model delivery via HTTP Range requests

### 4. Generated Font Licensing
**Decision:** All generated fonts must be OFL (Open Font License) licensed
- Training data: OFL-only Google Fonts
- Output licensing: OFL, enabling user commercial use
- Consistency: OFL training → OFL output
