# Major — ML Engineering (2026-02-25T112635)

## Scope
Created ML training pipeline for Cyrillic font generation.

## Deliverables
- `src/model/requirements.txt` — PyTorch, torchvision, onnx, onnxruntime dependencies
- `src/model/train_config.yaml` — training hyperparameters, model architecture specs
- `src/model/dataset.py` — OFL Google Fonts loader, augmentation pipeline
- `src/model/download_fonts.py` — fetches paired Latin+Cyrillic OFL fonts
- `src/model/model.py` — conditional GAN: StyleEncoder + UNetGenerator
- `src/model/train.py` — training loop, checkpointing
- `src/model/export_onnx.py` — ONNX export with dynamic quantization (opset 17)
- `src/model/inference_contract.md` — browser inference spec
- `data/README.md` — dataset directory docs
- `models/README.md` — model artifact directory docs

## Key Decisions
- **Russian charset:** 66 chars (А–Я + а–я with Ё/ё)
- **Style glyphs:** Latin A,B,H,O,g,n,o,p,s,x (10 chars, max structural diversity)
- **ONNX inputs:** style_glyphs [B,10,1,128,128] float32, char_index [B] int64
- **ONNX output:** generated_glyph [B,1,128,128] float32 ([-1,1] range)
- **Training data:** OFL-only Google Fonts with both Latin+Cyrillic
- **Browser inference:** WebGL preferred (~15–30ms/glyph), WASM fallback (~80–150ms/glyph)
