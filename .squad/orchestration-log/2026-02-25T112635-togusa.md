# Togusa — Frontend (2026-02-25T112635)

## Scope
Created React+Vite frontend for browser-based Cyrillic font generation.

## Deliverables
- `src/frontend/package.json` — React, TypeScript, Vite, onnxruntime-web deps
- `src/frontend/vite.config.ts` — WASM asset handling, backend proxy
- `src/frontend/tsconfig.json` — strict TypeScript config
- `src/frontend/App.tsx` — main app, layout
- `src/frontend/appStore.ts` — Zustand state management
- `src/frontend/OnnxInference.ts` — ONNX Runtime Web wrapper, InferenceSession
- `src/frontend/FontLoader.ts` — uploaded font parsing
- `src/frontend/cyrillicCharset.ts` — Russian charset (66 chars)
- `src/frontend/components/FontUpload.tsx` — file upload UI
- `src/frontend/components/GlyphPreview.tsx` — generated glyph canvas
- `src/frontend/components/ModelLoadingBar.tsx` — progress indicator

## Key Decisions
- **Framework:** React 18 + TypeScript + Vite
- **ML runtime:** ONNX Runtime Web (WASM + WebGL)
- **Style glyphs:** Latin A,B,C,D,E,H,I,O,R,X at 128x128 grayscale
- **Charset:** 66 Russian chars (33 upper, 33 lower), indices 0–65
- **Model loading:** fetch with progress tracking, InferenceSession persistence
- **Dev server:** Vite proxy for /api → backend :5000
