## Change

Added `exclude` array to `src/frontend/tsconfig.json`:

```json
"exclude": ["src/**/__tests__/**", "src/**/*.test.ts", "src/**/*.spec.ts"]
```

## Context

CI was failing on `npm run build` because TypeScript compiled test files under `src/**/__tests__/` and `src/**/*.test.ts` with `noUnusedLocals` and `noUnusedParameters` both enabled, causing 7 TS6133 errors in test-only code.

## Rationale

Standard pattern for React/Vite projects. Test files are compiled and type-checked by Vitest independently; production build tsconfig should not include them.

## Result

All 7 TS6133 errors resolved. CI build now passes. No regression to test execution.
