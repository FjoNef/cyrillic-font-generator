# Team

## Project Context

- **Project:** Cyrillic Font Generator
- **Description:** A web app that generates Cyrillic font symbols for non-Cyrillic fonts using AI. The pre-trained AI model is shipped to the client and all generative work executes in the browser (client-side inference). .NET backend.
- **Stack:** .NET backend, Web frontend (TBD: Blazor/WASM or React), ONNX/WebAssembly for client-side AI inference
- **User:** FjoNef
- **Universe:** Ghost in the Shell
